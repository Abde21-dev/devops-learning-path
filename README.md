# DevOps Learning Path 🚀

Parcours d'apprentissage DevOps autodidacte en préparation d'une formation POEI DevOps Cloud.

## Objectif

Acquérir les bases solides pour intégrer un poste DevOps Cloud :
Linux, Git, Docker, CI/CD, et Cloud.

## Programme

### Module 1 — Linux ✅
- Navigation et arborescence
- Permissions (chmod, chown)
- Gestion de fichiers (grep, find, head, tail)
- Services système (systemctl, journalctl)
- Variables d'environnement
- Scripting bash

### Module 2 — Git 🔄 (en cours)
- Configuration et initialisation
- Workflow : add, commit, push, pull
- Branches et merge
- Résolution de conflits

### Module 3 — Docker ⏳
- Images et conteneurs
- Dockerfile
- Docker Compose
- Volumes et réseaux

### Module 4 — CI/CD ⏳
- Concepts pipeline
- GitHub Actions
- Tests automatisés

### Module 5 — Cloud ⏳
- Notions AWS/GCP
- Infrastructure as Code
- Terraform bases

## Scripts produits

| Script | Description |
|--------|-------------|
| `rapport-systeme.sh` | Rapport santé serveur : disque, RAM, processus, services |
| `verification.sh` | Vérification état services et fichiers critiques |
